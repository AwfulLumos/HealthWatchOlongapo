# HealthWatch Olongapo - User Guide

## Getting Started

### System Login

The HealthWatch Olongapo system requires authentication to access patient data and clinical features.

**Demo Login Credentials:**

| Role | Username | Password | Access Level |
|------|----------|----------|-------------|
| Admin | `admin` | `admin123` | Full system access |

### First Login
1. Open the HealthWatch Olongapo application in your web browser
2. Enter your username and password

## Patient Management

### Viewing Patient List
1. Navigate to **Patients** from the sidebar
2. View all registered patients in the table
3. Use the search box to find patients by name or ID
4. Filter by barangay using the dropdown menu
5. Sort columns by clicking column headers

### Adding a New Patient
1. Click **"Add Patient"** button
2. Fill in the patient registration form:
   - **Personal Information**: First Name, Last Name, Date of Birth, Gender
   - **Contact Details**: Phone number, Address, Emergency Contact
   - **Medical Information**: Blood Type, Civil Status
   - **Location**: Barangay (neighborhood)
   - **Insurance**: PhilHealth ID number
3. Click **"Save Patient"** to register

### Editing Patient Information
1. Find the patient in the list
2. Click the **Edit** icon (pencil) in the Actions column
3. Modify the necessary information in the form
4. Click **"Update Patient"** to save changes

### Viewing Patient Details
1. Click the **View** icon (eye) in the Actions column
2. Review complete patient information
3. See medical history and contact details
4. Close the modal when finished

## Appointment Management

### Scheduling New Appointments
1. Go to **Appointments** section
2. Click **"Schedule Appointment"**
3. Fill in appointment details:
   - **Patient**: Select from dropdown or type to search
   - **Staff/Doctor**: Choose assigned healthcare provider
   - **Date & Time**: Select appointment slot
   - **Purpose**: Reason for the visit
   - **Notes**: Additional information (optional)
4. Click **"Schedule"** to confirm

### Managing Existing Appointments
1. View appointments in **List View** or **Calendar View**
2. Use filters to find specific appointments:
   - Filter by status (All, Confirmed, Pending, Cancelled, Completed)
   - Search by patient name
3. Edit appointments by clicking the Edit icon
4. Update appointment status as needed:
   - **Confirmed** - Appointment verified
   - **Pending** - Waiting for confirmation
   - **Cancelled** - Appointment cancelled
   - **Completed** - Patient visit finished

### Calendar View
- Switch to Calendar View for visual scheduling
- See daily appointment load
- Identify available time slots
- Click on calendar dates to view day's appointments

## Consultation Management

### Recording Consultations
1. Navigate to **Consultations**
2. Click **"Add Consultation"**
3. Complete the consultation form:
   - **Patient Information**: Select patient from dropdown
   - **Staff/Doctor**: Assign healthcare provider
   - **Date**: Consultation date
   - **Chief Complaint**: Patient's main concern
   - **Symptoms**: Detailed symptom description
   - **Diagnosis**: Medical diagnosis
   - **ICD Code**: International Classification of Diseases code
   - **Type**: Regular, Follow-up, or Emergency
   - **Status**: Completed, In Progress, or Referred
   - **Notes**: Additional clinical notes
4. Click **"Save Consultation"**

### Viewing Consultation History
1. Browse all consultations in the table
2. Filter by consultation type or status
3. Search by patient name or consultation ID
4. Click View to see detailed consultation records
5. Edit consultations if corrections are needed

## Prescription Management

### Issuing Prescriptions
1. Go to **Prescriptions** section
2. Click **"Add Prescription"**
3. Enter prescription details:
   - **Consultation**: Link to related consultation (optional)
   - **Patient**: Select patient receiving prescription
   - **Doctor**: Prescribing physician
   - **Date**: Prescription date
   - **Medicine**: Medication name
   - **Dosage**: Amount per dose (e.g., "500mg")
   - **Frequency**: How often to take (e.g., "3 times daily")
   - **Duration**: Treatment period (e.g., "7 days")
   - **Instructions**: Special directions for patient
4. Click **"Save Prescription"**

### Managing Prescriptions
1. View all prescriptions in the list
2. Search by patient name or prescription ID
3. Filter by doctor or date range
4. Edit prescriptions for corrections
5. Print or share prescription details with patients

## Vital Signs Recording

### Recording Patient Vital Signs
1. Navigate to **Vital Signs**
2. Click **"Add Vital Signs"**
3. Enter measurements:
   - **Patient**: Select patient from dropdown
   - **Date**: Measurement date
   - **Blood Pressure**: Systolic/Diastolic (e.g., 120/80 mmHg)
   - **Pulse Rate**: Beats per minute
   - **Respiration Rate**: Breaths per minute
   - **Temperature**: Body temperature in Celsius
   - **Blood Sugar**: mg/dL measurement
   - **Weight**: Patient weight in kilograms
   - **Height**: Patient height in centimeters
   - **BMI**: Automatically calculated from weight and height
4. Click **"Save Vital Signs"**

### Tracking Vital Signs History
1. View all vital signs measurements in the table
2. Search by patient name to see individual trends
3. Monitor changes over time for chronic conditions
4. Use data for clinical decision-making
5. Edit measurements if corrections needed

## Staff Management

### Adding Staff Members
1. Go to **Staff** section
2. Click **"Add Staff"**
3. Complete staff registration:
   - **Personal Information**: Full name, contact details
   - **Professional Details**: Role, license number, specialization
   - **Work Assignment**: Barangay station, schedule
   - **Account Settings**: Username, password, role permissions
4. Click **"Save Staff"** to create account

### Managing Staff Accounts
1. View all staff members in the table
2. Filter by role (Doctor, Nurse, Midwife, BHW)
3. Search by name or staff ID
4. Update staff information using Edit button
5. Activate/deactivate staff accounts as needed

### Staff Roles and Permissions
- **Doctor**: Full clinical access, prescription authority
- **Nurse**: Patient care, vital signs, basic consultations
- **Midwife**: Maternal and child health services
- **BHW (Barangay Health Worker)**: Community health, basic patient care
- **Admin**: Full system access, user management

- Data persists between sessions
- Clear browser cache will remove all data
- Export important data regularly (feature planned)

### Backup Considerations
- **Browser Storage**: Data stored in localStorage
- **Multiple Devices**: Data is device-specific
- **Data Loss Prevention**: Avoid clearing browser cache
- **Regular Backups**: Plan for periodic data export (future feature)

## Troubleshooting

### Common Issues

**Cannot Login:**
1. Verify username and password are correct
2. Check if account is active
3. Try refreshing the page
4. Clear browser cache if persistent issues

**Data Not Saving:**
1. Check browser localStorage is enabled
2. Ensure sufficient storage space
3. Try refreshing the page
4. Check for browser console errors

**Performance Issues:**
1. Close unnecessary browser tabs
2. Clear browser cache periodically
3. Use latest browser version
4. Check available system memory

**Missing Navigation:**
1. Refresh the page
2. Log out and log back in
3. Check user role permissions
4. Contact administrator if persistent

## Best Practices

### Daily Workflow
1. **Start with Dashboard** - Check daily statistics and upcoming appointments
2. **Update Patient Records** - Keep patient information current
3. **Record Consultations** - Document all patient visits promptly
4. **Manage Appointments** - Update statuses and schedule new appointments
5. **Enter Vital Signs** - Record measurements during patient visits

### Data Quality
- **Complete Information**: Fill all required fields
- **Accurate Data**: Double-check patient information and medical data
- **Timely Updates**: Record information promptly after patient visits
- **Consistent Formats**: Use standard formats for dates, measurements, etc.

### Security Best Practices
- **Secure Passwords**: Use strong passwords and change regularly
- **Logout Properly**: Always log out when finished
- **Private Browsing**: Use on shared computers
- **Data Privacy**: Follow HIPAA and local privacy guidelines for patient data
